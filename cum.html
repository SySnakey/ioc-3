Cum intrii pe serverul meu?

Nimic mai simplu..

  • Pasul 1

    • Descarci si instalezi TLauncher - link download aici

  • Pasul 2

    • Deschideti TLauncher, mergeti la 'Versios' in partea de jos-mijlooc si selectati optiunea de 'Create version'

    Selectati urmatoarele:

      Name: scrieti VictorYT
      Release type: Forge
      Game Version: 1.16.5
      Version name: 36.2.39
      [Optional] Bifati 'Add OptiFine'; Debifati 'Go to TL Mods catalog' si 'Use the skins system TLouncher.org'
      Lasati restul asa cum este si dati click pe create
      Dupa dati click pe 'Install' in partea dreapta de versiune. Dupa ce s-a instalat, apasati pe 'play' si asteptati sa se deschida jocul. Odata deschis, dati exit/iesiti din joc

  • Pasul 3

    • Dati click pe 'Folders' (icoada care arata ca un folder din partea dreapta jos. In fereastra care s-a deschis, intrati in folderul 'version' iar inauntrul acestuia veti vedea un folder numit 'VictorYT' - intrati si in acest folder iar dupa, inauntru in folderul 'mods'

  • Pasul 4

    • Descarcati mod-urile pentru acest server de aici

    Odata descarcate, mergeti in locatioa unde le-ati descarcat, dezarhivatile (click dreapta - extract here), intrati in folderul nou creat numit 'moduri', selectati toate fiserele dinantru (click stanga pe unul singur si dupa apasati CTRL+A), dati click dreapta dupa ce toate sunt selectate si selectati 'copy'

    Mergeti in folderul 'mods' deschis la pasul 3 si dati click dreapta in acest folder, pe alb si selectati 'paste'

  • Pasul 5

    • Intrati in joc folosind TLauncher si versiunea nou creata (VictorYT), mergeti pe 'Multiplayer' si dati add-server (daca nu este deja in lista serverul VictorYT)

    La adresa serverului treceti: VictorYT.top iar la restul lasati asa cum este, si dati 'add'

    Odata ce aveti serverul in lista, intrati pe server (dublu-click) and ENJOY :)
